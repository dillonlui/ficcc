03 · Typography
Type scale with palette
Lora for headlines and pull quotes — warm, sturdy, and bookish. DM Sans for body, UI, and navigation. Noto Serif SC and Noto Sans SC for CJK, explicitly declared to avoid inconsistent system fallbacks across platforms.
Font stack reference
Serif (headlines, quotes): Lora → Noto Serif SC → Georgia → serif
Sans (body, UI, nav): DM Sans → Noto Sans SC → system-ui → sans-serif
CJK-only contexts: Use --font-serif-cjk or --font-sans-cjk to skip Latin fonts
Performance note: Noto CJK fonts are large (~4MB). Subset to characters used, or lazy-load after Latin fonts.

Bilingual layout guidance
Stacked pattern (preferred for headlines): Chinese on top, English below at a slightly smaller size or lighter color. This avoids line-height conflicts between scripts.

Inline pattern (for labels and short text): Works for ≤6 Chinese characters + short English. Example: "主日崇拜 Sunday Worship." Avoid for anything longer — the mixed scripts create uneven rhythm.

Line-height note: CJK characters sit taller in the line box. Use line-height: 1.8 for CJK body copy vs 1.7 for Latin body copy to maintain visual consistency.

Language toggle vs. dual display: For content-heavy pages, prefer a language toggle over duplicating both languages inline. Reserve dual display for service info, event listings, and navigation where both audiences need quick scanning.

06 · Accessibility
Contrast checks
Key pairings tested against WCAG 2.1 AA (4.5:1 for normal text, 3:1 for large text and UI components).

Passing combinations

Base 700 on 50 7.4:1 AAA

White on Primary 500 5.8:1 AA
Base 100 on Primary 700 10.1:1 AAA
White on Secondary 400 3.4:1 AA-lg
Base 200 on 900 11.8:1 AAA
Base 600 on 50 4.7:1 AA

Note: Secondary 400 (baked clay) on white is AA for large text only. Use Secondary 600+ for small text on light backgrounds.
